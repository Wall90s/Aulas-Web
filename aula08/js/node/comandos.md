# API com NODE + Express

## Instalação do Express
NPM - Node Package Manager (Gerenciador de pacotes Node)<br>
``npm install express``
<br>

## Métodos HTPP
| **Crud** | **HTTP** | **SQL** |
|:--------:|:--------:|:-------:|
|  create  |   post   |  insert |
|   read   |    get   |  select |
|  update  |    put   |  update |
|  delete  |  delete  |  delete |
