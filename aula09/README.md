# Criando sistemas Backend e Frontend

### Backend - NODE.JS (API)
- NODE
- Express
    <br> ``npm init -y``
    <br> ``npm install express``
    <br> ``npm install cors``
### Frontend - REACT.JS
- REACT - SPA (Single Page Application)
    <br> ``npx create-react-app "nome da aplicação"``
    <br> ``cd app``
    <br> ``npm start``
    <br> ``npm install axios``
